Используя Python и FastAPI, создай  API (размести в директорий core) для упаковки изображений в cbz с добавлением метаданных манги/книги/комикса.

Сервис должен ориентироваться на 2 существующие в системе директорий (исходники и запакованные издания), путь к которым указан в конфигураций. Содержимое директорий должно автоматически пересканироваться со временем.

В директорий с исходники ожидаются директорий с изображениями.

В директорию с изданиями будут попадать cbz архивы, собранные из одной из директорий в источниках.

CBZ архив должен содержать изображения и ComicInfo.xml в стандартном для комиксов формате.

API должно содержать следующие методы:
* Принудительно пересканировать директорию (исходники или издания)
* Получение списка обнаруженных директорий в источниках
* Получение списка существующих изданий
* Метод редактирование метаданных уже упакованного издания
* Метод сборки изображений из директорий в издание с добавлением к нему метаданных
* Загрузка новой директорий в источники (с массивом файлов)
* Загрузка нового издания в директорию изданий (cbz файл)
* Удаление директорий в исходниках
* Удаление файла из директорий в исходниках
* Удаление издания
* Переименование директорий в исходниках и файлов в ней содержащихся
* Получение содержимого издания (изображения и метаданные)
* Методы получения списка уже встречавшихся авторов/тегов/жанров

Сервис должен иметь frontend на React (размести в директорий frontend)

Подготовь Dockerfile для сборки образа с nginx, который будет разворачивать собранный фронтэнд и бэкэнд внутри себя.

Dockerfile должен:
* Выводить наружу порты 80 (HTTP) и 443 (TLS)
* Создавать тома для директорий с исходниками, директорий с изданиями и директорий с конфигурационными файлами.
